REaCT

I have redesigned the mobile application for our capstone project, called REaCT (Recognize, Eliminate, and Contact Trace) is a Covid-19 Contact Tracing Application. It aims to assist in contact tracing within various establishments by utilizing Facial Recognition Technology (FRT). This application serves as the front-end mobile application that primarily functions as the registration system for REaCT.

Features:
  • Account Registration
  • Facial Recognition
  • Preview COVID-19 status of the City of Dagupan
  • View places you have visited from the past 14 days
  • Change account information and password
  • Send out an alert if you are COVID-positive or negative
